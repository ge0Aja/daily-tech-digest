## AI & Machine Learning

### Anthropic's Claude Mythos Finds Thousands of Zero-Days via Project Glasswing
**Source:** Anthropic | [Read more](https://www.anthropic.com/glasswing)

Anthropic revealed Claude Mythos Preview, a frontier model that autonomously discovered thousands of zero-day vulnerabilities across every major OS and browser, including a 17-year-old unauthenticated RCE in FreeBSD (CVE-2026-4747). Rather than a public release, Anthropic launched Project Glasswing — a $100M initiative granting exclusive model access to 40+ security partners including AWS, Apple, Microsoft, and Google. The model can identify and exploit vulnerabilities end-to-end; Anthropic is using it defensively to patch critical infrastructure. This raises urgent questions about AI-powered offense and whether controlled distribution is a durable safety strategy.

### EU AI Act High-Risk Compliance Deadline: August 2026
**Source:** Holistic AI | [Read more](https://www.holisticai.com/blog/ai-regulation-in-2026-navigating-an-uncertain-landscape)

The EU AI Act's high-risk AI provisions become enforceable in August 2026, covering AI used in hiring, credit, medical, and law enforcement applications. A proposed amendment may extend this to December 2027, but it is not finalized. US companies operating in the EU must comply regardless of domestic deregulation — the rules apply extraterritorially. Engineers building AI for regulated use cases should treat compliance as a first-class design constraint now rather than a downstream legal task.

## Security & Privacy

### Microsoft April 2026 Patch Tuesday: 167 CVEs, 2 Zero-Days
**Source:** Bleeping Computer | [Read more](https://www.bleepingcomputer.com/news/microsoft/microsoft-april-2026-patch-tuesday-fixes-167-flaws-2-zero-days/)

Microsoft patched 167 vulnerabilities including 8 Critical and 2 zero-days this month. CVE-2026-32201, an actively exploited SharePoint spoofing flaw (CVSS 6.5), and CVE-2026-33825, a Windows Defender privilege escalation with public exploit code, are the most urgent. The worst new issue is CVE-2026-33824, an unauthenticated RCE in Windows IKE Service Extensions (CVSS 9.8). Security teams should prioritize SharePoint and IKE patches immediately given active exploitation in the wild.

### Apple Expands DarkSword Exploit Patches to iOS 18.7.7
**Source:** Malwarebytes | [Read more](https://www.malwarebytes.com/blog/news/2026/04/apple-expands-darksword-patches-to-ios-18-7-7)

Apple extended DarkSword exploit kit patches to all iPhone 11 through 16 models via iOS 18.7.7. DarkSword chains six vulnerabilities across WebKit, Safari, the dynamic loader, and the kernel — enabling full device compromise from a single visit to a malicious webpage. The kit has been used in targeted attacks in Saudi Arabia, Turkey, Malaysia, and Ukraine since July 2025. Apple backporting patches within iOS 18 without requiring an OS upgrade is a notable policy shift.

### CISA Adds 7 Actively Exploited Flaws to KEV Catalog
**Source:** CISA | [Read more](https://www.cisa.gov/news-events/alerts/2026/04/13/cisa-adds-seven-known-exploited-vulnerabilities-catalog)

CISA added seven vulnerabilities to its KEV catalog on April 13, including Fortinet FortiClient EMS CVE-2026-21643 (SQL injection, CVSS 9.1, unauthenticated RCE), Adobe Acrobat CVE-2026-34621 (prototype pollution), and Microsoft Exchange and Windows CLFS driver flaws. Federal agencies must patch FortiClient EMS by April 16 and the rest by April 27. The FortiClient flaw allows unauthenticated code execution and should be treated as an emergency patch across enterprise environments.

## Cloud & Infrastructure

### MySQL 8.0 Reaches End of Life — Migrate to 8.4 LTS Now
**Source:** The Register | [Read more](https://www.theregister.com/2025/10/31/mysql_8_support_end/)

MySQL 8.0 officially hits end of life this month, ending security patches and bug fixes after eight-plus years. The migration path is MySQL 8.4 LTS (supported through 2032), but it is not a drop-in upgrade — `mysql_native_password` is disabled by default in 8.4, requiring auth plugin migration. AWS RDS extends 8.0 support only until July 2026; Aurora until April 2028. Running unsupported MySQL in production is a direct PCI DSS and HIPAA compliance violation.

**Related:** [Oracle EOL announcement](https://blogs.oracle.com/mysql/endoflife-of-heatwave-mysql-80-version) | [Migration guide (JusDB)](https://www.jusdb.com/blog/mysql-80-eol-in-2026-why-upgrading-to-mysql-84-lts-is-mission-critical)

### Oracle Ships Multi-Region Distributed Database with Async Raft Replication
**Source:** Oracle Database Blog | [Read more](https://blogs.oracle.com/database/from-ha-to-multi-region-always-on-distributed-databases-with-asynchronous-raft-replication)

Oracle announced its Globally Distributed AI Database supporting multi-region resilience via asynchronous Raft replication in an active-passive topology. The design avoids the write-latency penalty of synchronous cross-region consensus while maintaining strong availability guarantees, positioning it competitively against Aurora DSQL and DynamoDB multi-region strong consistency. This continues a broad industry convergence on Raft as the default mechanism for globally distributed OLTP.

## Industry & Trends

### Open Source After Mythos: A Reckoning for Community Security
**Source:** IBM Newsroom | [Read more](https://newsroom.ibm.com/2026-04-09-Open-Source,-After-Mythos)

Following Anthropic's Mythos disclosure, IBM published analysis on what AI-powered vulnerability discovery means for open source. If frontier models can find zero-days in community codebases faster than volunteers can patch them, the "many eyes" security model faces structural strain. The post argues that AI-assisted vulnerability scanning must become a standard practice for open source projects. Anthropic's $2.5M donation to OpenSSF is cited as a necessary but insufficient first step.

### Vulnerability Exploits Overtake Phishing as Top Initial Access Vector
**Source:** Cisco Talos | [Read more](https://blog.talosintelligence.com/microsoft-patch-tuesday-april-2026/)

Cisco Talos data shows vulnerability exploits accounted for nearly 40% of Q4 2025 intrusions, surpassing phishing as the most common initial access method for the first time. AI-assisted exploit tooling has compressed the window between CVE disclosure and a working exploit from weeks to days. This reinforces the urgency of rapid patch cadence and continuous asset exposure management — organizations relying on detection over prevention are increasingly exposed.
