**Source:** Google Blog | **URL:** https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/

### Google Gemini Diffusion Released in Beta
**Source:** Google | **URL:** https://llm-stats.com/ai-news

Google announced Gemini Diffusion as a beta release in May 2026, marking an architectural departure from autoregressive token generation. Diffusion-based language models generate text by progressively denoising a sequence rather than predicting one token at a time, offering potential speed and parallelism advantages. The release adds to a growing wave of architectural experimentation at the frontier, challenging the assumption that transformer-decoder autoregression is the only viable path for capable language models.

## Security & Privacy

### CVE-2026-31431 "Copy Fail": 9-Year Linux Kernel Flaw Enables Reliable Root Escalation
**Source:** Help Net Security / Microsoft Security Blog | **URL:** https://www.helpnetsecurity.com/2026/04/30/copyfail-linux-lpe-vulnerability-cve-2026-31431/

A logic flaw in the Linux kernel's AF_ALG cryptographic API module (present in all major distros since 2017) lets any unprivileged local user gain root access using a deterministic 732-byte Python script — no race conditions, retries, or timing windows required. The exploit corrupts a setuid binary in the page cache without touching disk, bypassing standard file-integrity checks, and also functions as a Kubernetes container escape primitive. CISA has added CVE-2026-31431 to its Known Exploited Vulnerabilities catalog with a public PoC available; patch your kernel or block AF_ALG socket creation immediately.

### ADT Data Breach Exposes 5.5M Customers via Okta Vishing Attack
**Source:** Bleeping Computer | **URL:** https://www.bleepingcomputer.com/news/security/home-security-giant-adt-data-breach-affects-55-million-people/

ShinyHunters compromised an ADT employee's Okta SSO account through voice phishing (vishing), then exfiltrated records from the company's Salesforce instance. Exposed data includes names, emails, dates of birth, phone numbers, addresses, and partial government IDs for 5.5 million people — though no payment information was accessed. ADT discovered the breach April 20, 2026; this is the company's third incident in under two years, illustrating the systemic risk of SSO credential campaigns that ShinyHunters runs against Microsoft Entra, Okta, and Google SSO accounts via BPO employees.

### Microsoft Patches Actively Exploited SharePoint Zero-Day (CVE-2026-32201) in 168-Vulnerability Patch Tuesday
**Source:** SecurityWeek / Tenable | **URL:** https://www.securityweek.com/microsoft-patches-exploited-sharepoint-zero-day-and-160-other-vulnerabilities/

April 2026 Patch Tuesday addressed 168 vulnerabilities, highlighted by CVE-2026-32201 — a spoofing flaw in SharePoint Server allowing unauthenticated network-based content forgery, already actively exploited in the wild before patching. Successful exploitation enables an attacker to display falsified content within trusted SharePoint environments, making phishing campaigns significantly harder to detect. CISA required Federal agencies to remediate by April 28; organizations running SharePoint 2016, 2019, or Subscription Edition should apply the update immediately.
