## Security & Privacy

### CVE-2026-31431 "Copy Fail" — Linux Kernel LPE Threatens Kubernetes Multi-Tenancy
**Source:** Microsoft Security Blog | [Read more](https://www.microsoft.com/en-us/security/blog/2026/05/01/cve-2026-31431-copy-fail-vulnerability-enables-linux-root-privilege-escalation/)

A CVSS 7.8 local privilege escalation flaw in the Linux kernel's AF_ALG crypto module allows unprivileged users to escalate to root via splice() system call abuse, affecting virtually all Linux distributions from kernel versions dating to 2017 — including Ubuntu 24.04 LTS, RHEL 10.1, Amazon Linux 2023, and SUSE 16. Particularly dangerous in multi-tenant Kubernetes environments where it can enable container breakout and host compromise. Patch immediately; if patches are unavailable, disable AF_ALG sockets and treat any container RCE as potential host-level compromise.

### ADT Suffers Third Breach in Two Years: 5.5M Customers Exposed via Okta Vishing
**Source:** Bleeping Computer | [Read more](https://www.bleepingcomputer.com/news/security/home-security-giant-adt-data-breach-affects-55-million-people/)

ShinyHunters used a voice phishing (vishing) attack to steal an ADT employee's Okta SSO credentials, then pivoted into the company's Salesforce CRM to extract records for 5.5 million customers — names, phone numbers, addresses, and partial SSNs. After ADT declined extortion demands, ShinyHunters leaked an 11GB archive on the dark web. The incident is a textbook SSO-to-SaaS lateral movement attack and ADT's third breach since 2024, reinforcing the case for phishing-resistant MFA on all identity providers.

### SharePoint Zero-Day CVE-2026-32201 Under Active Exploitation, Added to CISA KEV
**Source:** The Hacker News | [Read more](https://thehackernews.com/2026/04/microsoft-issues-patches-for-sharepoint.html)

Microsoft's April 2026 Patch Tuesday fixed a SharePoint Server spoofing zero-day (CVE-2026-32201, CVSS 6.5) already under active exploitation, alongside 167 other CVEs. The flaw allows network-adjacent attackers to spoof trusted interfaces via improper input validation, impacting confidentiality and integrity. CISA added it to the Known Exploited Vulnerabilities catalog with a federal remediation deadline of April 28; unpatched SharePoint Server deployments remain at immediate risk.

### AI-Assisted Attacks Surge: Time to Exploit Collapses to 44 Days
**Source:** The Hacker News | [Read more](https://thehackernews.com/2026/05/2026-year-of-ai-assisted-attacks.html)

The median window between vulnerability disclosure and active exploitation has shrunk from over 700 days in 2020 to just 44 days, driven by AI tools that automate exploit development. Security teams that rely on manual patch-and-detect cycles now face a structurally inadequate response window.
