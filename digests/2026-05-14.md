## AI & Machine Learning

### Google Detects First AI-Generated Zero-Day Exploit
_CNBC / Google Threat Intelligence_ | [Read more](https://www.cnbc.com/2026/05/11/google-thwarts-effort-hacker-group-use-ai-mass-exploitation-event.html)

Google's Threat Intelligence Group confirmed the first known zero-day exploit built using AI — a 2FA bypass crafted by a criminal threat actor who planned a mass exploitation event before Google disrupted the campaign. The exploit was identified as AI-generated based on telltale characteristics: overly explanatory comments in the code, a fabricated severity rating, and Python patterns common in LLM-generated output. Google says it does not believe its own Gemini model was used. This marks a watershed moment: AI-assisted zero-day development has moved from theoretical concern to confirmed, real-world offensive capability.

### Yann LeCun's AMI Labs Raises $1.03B to Build World Models
_TechCrunch_ | [Read more](https://techcrunch.com/2026/03/09/yann-lecuns-ami-labs-raises-1-03-billion-to-build-world-models/)

AMI Labs, the Paris-based AI startup co-founded by Turing Award winner Yann LeCun after leaving Meta, closed the largest seed round in European history at $1.03B, valuing the company at $3.5B. AMI is building world models based on LeCun's Joint Embedding Predictive Architecture (JEPA) — an alternative to LLMs that learns by modeling how the physical world works rather than predicting next tokens, targeting industrial, robotics, and healthcare domains where LLM limitations are most consequential. The company is spending its entire first year on R&D before shipping any product.

## Security & Privacy

### CVE-2026-31431 "Copy Fail": Linux Kernel Root Escalation — CISA Deadline May 15
_Microsoft Security Blog / Help Net Security_ | [Read more](https://www.microsoft.com/en-us/security/blog/2026/05/01/cve-2026-31431-copy-fail-vulnerability-enables-linux-root-privilege-escalation/)

Copy Fail is a nine-year-old logic bug in the Linux kernel's `algif_aead` cryptographic module that allows any unprivileged local user to gain root via a 732-byte Python exploit — leaving no disk trace and evading file-integrity monitoring. The flaw enables container breakout, multi-tenant compromise, and lateral movement across Kubernetes, CI/CD, and shared cloud environments. CISA added it to the Known Exploited Vulnerabilities catalog with a federal remediation deadline of May 15. Patch your kernel packages immediately on all affected distributions (RHEL, Ubuntu, Amazon Linux, SUSE, Debian, Arch) or block AF_ALG socket creation as a stopgap.

### Canvas LMS Breach: ShinyHunters Hits 275M Users Across 9,000 Institutions
_The Hacker News_ | [Read more](https://thehackernews.com/2026/05/instructure-reaches-ransom-agreement.html)

ShinyHunters breached Instructure (parent of Canvas LMS) twice — on April 29 and again May 7 — stealing 3.65TB of data on an estimated 275 million students and teachers at 8,809 universities worldwide, including names, email addresses, student IDs, and private messages. The group disrupted platform access during US finals week and threatened to publish the full dataset before Instructure paid a ransom and received "shred logs" as confirmation of destruction. This is the largest educational data breach on record and a stark reminder that LMS platforms carry sensitive PII at extraordinary scale.

_The Hacker News_ | [Read more](https://thehackernews.com/2026/05/microsoft-patches-138-vulnerabilities.html)

Microsoft's May 2026 Patch Tuesday fixes 138 CVEs, with 30 rated Critical. The two most severe are CVE-2026-41096 (CVSS 9.8), a heap-based buffer overflow in the Windows DNS client enabling unauthenticated remote code execution over the network, and CVE-2026-41089, a stack-based buffer overflow in Windows Netlogon allowing unauthenticated code execution against domain controllers via a crafted network packet. No zero-days are in this batch, but both RCEs require immediate prioritization — especially on internet-facing systems and domain controllers.

### VS Code 1.119: AI Agents Share Browser Tabs and Emit OpenTelemetry Traces
_Visual Studio Magazine_ | [Read more](https://visualstudiomagazine.com/articles/2026/05/07/vs-code-1-119-adds-agent-browser-sharing-opentelemetry-tracing.aspx)

VS Code 1.119 (released May 6) ships two significant agent-facing capabilities. First, agents can now request shared access to specific browser tabs, enabling read and interaction with live pages from within the IDE, with user approval controls. Second, Copilot Chat agent sessions now emit OpenTelemetry traces, metrics, and events conforming to GenAI semantic conventions, pipeable to any OTLP-compatible backend for monitoring latency and token usage. Both features push VS Code toward treating AI agents as observable, first-class runtime processes.

_Microsoft Open Source Blog_ | [Read more](https://opensource.microsoft.com/blog/2026/04/02/introducing-the-agent-governance-toolkit-open-source-runtime-security-for-ai-agents/)
