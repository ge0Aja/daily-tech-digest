**Source:** Picus Security | [Read more](https://www.picussecurity.com/resource/blog/nginx-rift-cve-2026-42945-critical-heap-buffer-overflow-vulnerability-explained)

A critical heap buffer overflow in NGINX's URL rewrite module — present in all versions 0.6.27 through 1.30.0, spanning 18 years — has been disclosed and is actively exploited in the wild. Proof-of-concept code is public and exploitation against honeypots has been confirmed. Apply F5/NGINX upstream patches immediately.
