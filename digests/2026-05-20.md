## Security & Privacy

### CVE-2026-31431 "CopyFail": Nine-Year-Old Linux Kernel Flaw Enables Reliable Root Escalation
**Source:** Help Net Security | [Read more](https://www.helpnetsecurity.com/2026/04/30/copyfail-linux-lpe-vulnerability-cve-2026-31431/)

Disclosed April 29 by security firm Theori, CopyFail (CVSS 7.8) is a local privilege escalation bug in the Linux kernel's `algif_aead` module — the AEAD socket interface of the kernel's userspace crypto AF_ALG API — present in kernels from 2017 onward. The exploit abuses the interaction between AF_ALG sockets and `splice()` to perform a deterministic 4-byte write into the kernel page cache of any readable file, enabling corruption of in-memory binaries (e.g. `/usr/bin/su`) without touching disk. The 732-byte Python PoC achieves reliable root on Ubuntu, RHEL, SUSE, Debian, Arch, Amazon Linux, and Fedora with no race conditions; container breakout and lateral movement are realistic follow-on risks. Immediate mitigation: `echo "install algif_aead /bin/false" >> /etc/modprobe.d/disable-algif.conf`.
