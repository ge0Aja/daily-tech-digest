## Security & Privacy

### Microsoft Defender Zero-Days Exploited in the Wild — CISA Orders Patch by June 3
**Source:** SecurityWeek | [Read more](https://www.securityweek.com/microsoft-patches-exploited-undefend-and-redsun-defender-zero-days/)

Two Defender flaws published as PoCs on GitHub by researcher "Nightmare Eclipse" are actively exploited: CVE-2026-41091 "RedSun" (CVSS 7.8, local privilege escalation to SYSTEM via mpengine.dll) and CVE-2026-45498 "UnDefend" (CVSS 4.0, DoS that silently disables Defender definition updates). CISA added both to its KEV catalog and set a June 3 patch deadline for FCEB agencies; Microsoft released fixes in Malware Protection Engine v1.1.26040.8 and Antimalware Platform v4.18.26040.7. The ability to selectively blind endpoint protection on demand is the most dangerous aspect — it opens a silent window for malware deployment and lateral movement without triggering alerts.

### SonicWall Gen6 SSL-VPN MFA Bypass Enables Ransomware — Patch Alone Is Not Sufficient
**Source:** ReliaQuest | [Read more](https://reliaquest.com/blog/threat-spotlight-vpn-exploitation-when-patched-doesnt-mean-protected)

CVE-2024-12802 in SonicWall Gen6 SSL-VPN allows attackers to fully bypass MFA by authenticating via the UPN login path, which can have MFA disabled independently from the SAM path — an architectural flaw invisible to standard monitoring. The firmware patch requires six additional manual post-update reconfiguration steps that standard patch workflows do not verify, leaving appliances appearing remediated while fully exploitable. Akira ransomware (responsible for 40%+ of ransomware claims per At-Bay 2026 InsurSec Report) is actively brute-forcing credentials against these devices; Gen6 reached end-of-life April 16 with no further patches planned, making migration the only long-term solution.
