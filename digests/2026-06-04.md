## Security & Privacy

### Microsoft Defender Zero-Days Actively Exploited — Patch by June 3
**Source:** The Hacker News | [Read more](https://thehackernews.com/2026/05/microsoft-warns-of-two-actively.html)

Two Microsoft Defender vulnerabilities are being exploited in the wild. CVE-2026-41091 (CVSS 7.8) is a symbolic-link privilege escalation flaw that grants SYSTEM access to local attackers; CVE-2026-45498 (CVSS 4.0) is a DoS bug that disables Defender's endpoint protection entirely, opening a window for malware deployment or data exfiltration. Both are patched in Antimalware Platform versions 1.1.26040.8 / 4.18.26040.7; CISA set a June 3 compliance deadline for federal agencies under BOD 22-01.

### VS Code Zero-Day: One Click Steals Your GitHub OAuth Token
**Source:** Bleeping Computer | [Read more](https://www.bleepingcomputer.com/news/security/vs-code-zero-day-lets-hackers-steal-github-tokens-in-one-click/)

A VS Code zero-day lets a malicious webview simulate keypresses in the editor, open the Command Palette, install an attacker-controlled extension, and exfiltrate the victim's GitHub OAuth token — triggered by a single link click. Researcher Ammar Askar published a full PoC on June 2 after MSRC declined to patch it. Microsoft says the issue is mitigated in GitHub.dev services, though the underlying extension-to-webview message mechanism remains exploitable on VS Code Desktop.

### Miasma Supply Chain Attack Compromises 32+ Red Hat npm Packages
**Source:** Wiz Blog | [Read more](https://www.wiz.io/blog/miasma-supply-chain-attack-targeting-redhat-npm-packages)

On June 1, Wiz Research found 32+ releases under the @redhat-cloud-services npm namespace (~80K weekly downloads combined) containing unauthorized malicious code. The attacker used a compromised Red Hat employee GitHub account to trigger GitHub Actions OIDC flows, publishing trojanized packages with valid SLSA provenance. The Miasma worm sweeps for AWS/GCP/Azure credentials, GitHub tokens, Kubernetes configs, SSH keys, .env files, and npm/PyPI publish tokens — then self-propagates to new hosts.

### Charter Communications: 13M Records Leaked by ShinyHunters After Ransom Refusal
**Source:** Bleeping Computer | [Read more](https://www.bleepingcomputer.com/news/security/charter-confirms-data-breach-after-shinyhunters-extortion-threat/)

A vishing attack on April 1 compromised a Charter employee's Microsoft Entra account, giving attackers access to export millions of customer records from Charter's Salesforce instance. ShinyHunters initially claimed 40–42M records stolen, then published at least 13M records after Charter refused to pay ransom — including names, emails, addresses, phone numbers, and CPNI data. The breach underscores vishing + identity provider compromise as an increasingly reliable initial access path into large enterprise SaaS environments.

### Unpatched Windows Search URI Flaw Leaks NTLMv2 Hashes — Microsoft Won't Fix
**Source:** The Hacker News | [Read more](https://thehackernews.com/2026/06/unpatched-windows-search-uri.html)

A researcher disclosed an unpatched Windows search:// URI handler vulnerability that forces machines to initiate SMB connections to attacker-controlled servers, leaking NTLMv2 hashes exploitable for relay attacks. The mechanism is technically identical to CVE-2026-33829 in the Snipping Tool, but Microsoft declined to patch it as below their severity threshold. Recommended mitigations: block outbound TCP/445 and TCP/139, enforce SMB signing, and disable NTLM where feasible.

### Google Patches 124 Android Flaws Including Actively Exploited Zero-Day
**Source:** SC Media | [Read more](https://www.scworld.com/brief/google-releases-june-android-security-patches-addressing-124-vulnerabilities-including-one-zero-day)

Google's June 2026 Android security bulletin patches 124 vulnerabilities including CVE-2025-48595 — a high-severity integer overflow in the Android Framework enabling local privilege escalation without user interaction on Android 14+. Google noted possible limited, targeted exploitation consistent with commercial spyware or nation-state tradecraft. Pixel devices receive patches immediately; other OEMs will lag by weeks to months.

### Weedhack: Minecraft Malware-as-a-Service Campaign Has Infected 116K+ Systems
**Source:** McAfee Labs | [Read more](https://www.mcafee.com/blogs/other-blogs/mcafee-labs/weedhack-minecraft-malware-as-a-service-campaign-research/)

Active since January 2026, the Weedhack MaaS campaign uses SEO poisoning and YouTube to distribute 3,820+ malicious JAR files across 240+ URLs, infecting 116,000 systems at 2–3K/day. The free tier steals credentials from 36 browsers, 56 browser-based crypto wallets, Discord, Steam, and Telegram; premium tiers add full remote access to screen, webcam, and filesystem. The 850-member Telegram operator base illustrates the industrialization and commoditization of consumer-targeting malware platforms.

## Cloud & Infrastructure

### Microsoft Launches Azure Linux 4.0 Preview and Azure Container Linux GA
**Source:** InfoQ | [Read more](https://www.infoq.com/news/2026/05/azure-linux-4-server/)

### GKE on AWS and Azure Enter Maintenance Mode, Shutdown March 2027
**Source:** Google Cloud | [Read more](https://docs.cloud.google.com/kubernetes-engine/security-bulletins)

Google has placed GKE on AWS and GKE on Azure in maintenance mode, with both multi-cloud Kubernetes offerings scheduled for shutdown on March 17, 2027. Teams relying on these services need to plan migrations to native EKS, AKS, or standard GKE. ## Industry & Trends

### Colorado Guts Its AI Anti-Discrimination Law Before It Ever Took Effect
**Source:** TechTimes | [Read more](https://www.techtimes.com/articles/316756/20260517/colorados-ai-anti-discrimination-law-dead-before-it-ever-took-effect-replaced-notice-only.htm)

Governor Polis signed SB 189 on May 14, replacing Colorado's original AI Act with a notice-only framework and delaying enforcement to January 1, 2027. The original law — the US's first requiring algorithmic impact assessments, risk management programs, and deployer duties of care against discrimination in high-stakes decisions — was stripped of all substantive requirements after industry pushback. The rollback signals the ongoing political difficulty of passing enforceable AI safety obligations at the US state level.

### Open Source in 2026: AI Code Floods, Funding Pressure, and Licensing Wars
**Source:** Linux Insider | [Read more](https://www.linuxinsider.com/story/open-source-in-2026-faces-a-defining-moment-177630.html)

The open source ecosystem faces compounding strain in 2026: Valkey reported a 500% spike in lines of code submitted driven by AI-assisted contributions of uneven quality, accelerating maintainer burnout. Redis, HashiCorp/Terraform, and Elasticsearch continue relicensing to source-available models, while 55% of organizations cite vendor lock-in avoidance as their top OSS driver.
