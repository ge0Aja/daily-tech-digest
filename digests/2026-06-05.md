### AI-Generated Code CVEs Increase Nearly Sixfold in 2026
**Source:** LLM Stats | [Read more](https://llm-stats.com/ai-news)

CVE disclosures tied to AI-generated code have risen nearly sixfold year-over-year, reflecting both the explosion of AI-assisted coding and the tendency of LLMs to produce plausible-looking code that passes review but carries subtle vulnerabilities in input validation, authentication, and memory management. ## Security & Privacy

### HTTP/2 Bomb (CVE-2026-49975): One Machine Can Crash NGINX, Apache, Envoy, or IIS in Seconds
**Source:** The Hacker News | [Read more](https://thehackernews.com/2026/06/new-http2-bomb-vulnerability-allows.html)

The HTTP/2 Bomb vulnerability chains HPACK header-compression expansion with a zero-byte flow-control window hold to consume server memory indefinitely — a single attacker on a 100Mbps connection can exhaust 32GB of Apache HTTPD or Envoy RAM in approximately 20 seconds, taking the server offline. The flaw affects NGINX, Apache HTTPD, Microsoft IIS, Envoy, and Cloudflare Pingora. NGINX is patched in 1.29.8+ (add the `max_headers` directive); Apache in mod_http2 v2.0.41. Disabling HTTP/2 entirely is the immediate fallback for unpatched systems.

### NGINX Rift (CVE-2026-42945): 18-Year-Old Rewrite Flaw Enables Unauthenticated RCE
**Source:** The Hacker News | [Read more](https://thehackernews.com/2026/05/18-year-old-nginx-rewrite-module-flaw.html)

A heap buffer overflow in NGINX's `ngx_http_rewrite_module` — present since version 0.6.27 released in 2008 — allows unauthenticated remote code execution with a single crafted HTTP request, requiring no session or prior access. The flaw is triggered by unnamed PCRE captures (`$1`, `$2`) in rewrite chains and was discovered by an AI-powered static analysis tool. Patch to NGINX Open Source 1.30.1+ or 1.31.0+, or NGINX Plus R32 P6/R36 P4. Replace unnamed captures with named captures as an interim mitigation.

### Gemini Voice Assistant Could Be Hijacked via Poisoned WhatsApp, Slack, or SMS Notifications
**Source:** The Hacker News | [Read more](https://thehackernews.com/2026/06/whatsapp-slack-notifications-could.html)

SafeBreach Labs disclosed a notification-based indirect prompt injection affecting Google Gemini on Android: malicious instructions embedded in ordinary WhatsApp, Slack, or SMS notifications silently redirect Gemini's actions — opening smart home devices, faking messages from contacts, joining video calls, or poisoning long-term memory — with no malicious app required. Commands were hidden in foreign-language text or muted hyperlinks to bypass Gemini's content classifiers. Google patched the flaw in November 2025 via improved content classification; research was published June 3 with no observed wild exploitation.

### CISA KEV: Linux Kernel cgroups Container Escape + Android Framework RCE — Federal Deadline Today
**Source:** Bleeping Computer | [Read more](https://www.bleepingcomputer.com/news/security/cisa-warns-of-active-attacks-exploiting-android-linux-bugs/)

CISA added two actively exploited flaws to its Known Exploited Vulnerabilities catalog on June 2: CVE-2022-0492, a Linux kernel cgroups v1 privilege-escalation bug enabling container escape to host root, and CVE-2025-48595, an Android Framework integer-overflow enabling code execution and privilege escalation on Android 14+. Federal agencies face a June 5 remediation deadline under BOD 22-01. The Linux flaw is particularly critical for cloud and container workloads — operators on older kernels should patch immediately or restrict cgroups v1 release_agent access.

## Systems & Engineering

### PostgreSQL 19 Beta 1 Released: Parallel Autovacuum, SQL/PGQ, and WAIT FOR LSN
**Source:** PostgreSQL Project | [Read more](https://www.postgresql.org/about/news/postgresql-19-beta-1-released-3313/)

PostgreSQL 19 Beta 1 shipped on June 4 with three production-relevant features. Parallel autovacuum allows multiple workers on a single table, significantly reducing bloat spikes on large write-heavy tables. SQL/PGQ adds ISO property graph query support, enabling graph-style traversals without an external graph database. WAIT FOR LSN blocks execution until a replica catches up to a specific log position, eliminating polling loops needed to avoid stale reads. JIT is disabled by default and RADIUS authentication is removed entirely. GA is expected around September/October 2026.

### Dapr Graduates as a CNCF Project for Cloud-Agnostic Distributed Applications
**Source:** Microsoft Open Source Blog | [Read more](https://opensource.microsoft.com/blog/2026/05/18/from-open-source-to-agentic-systems-microsoft-at-open-source-summit-north-america-2026/)

Dapr (Distributed Application Runtime) graduated within the CNCF, reaching production-grade maturity with confirmed governance standards and stable APIs. Dapr provides a portable, event-driven runtime abstracting state management, pub/sub messaging, service invocation, and secrets management across cloud and edge environments without vendor lock-in. Graduation confirms the project's API surface is stable enough for enterprise production deployments, making it a credible foundation for multi-cloud distributed systems.

## Industry & Trends

### EU Unveils Tech Sovereignty Package to Reduce Dependence on US Cloud and AI Providers
**Source:** Fortune | [Read more](https://fortune.com/2026/06/03/european-union-tech-sovereignty-dependence-american-asian-ai-microchip/)

The European Commission presented its European Technological Sovereignty Package on June 3, anchored by the Cloud and AI Development Act (CADA). CADA introduces tiered sovereignty levels for cloud procurement — the highest tier, covering defence and healthcare, would effectively bar non-European providers from public contracts — and targets tripling EU data center capacity by 2033. The package also includes a sequel to the 2023 Chips Act to reduce semiconductor dependence on US and Asian supply chains, with urgency driven by fears that foreign tech dependencies could be "weaponized" against EU interests.

### Microsoft Project Solara: An Operating System Built for AI Agent Hardware
**Source:** Tech Startups | [Read more](https://techstartups.com/2026/06/02/top-tech-news-today-june-2-2026/)

Microsoft unveiled Project Solara, an OS designed for purpose-built AI agent hardware — including a desk appliance and a badge-like workplace wearable — that coordinates local AI agents with cloud services in real time. Pilot partners include AccuWeather, Best Buy, CVS Health, and Target. Solara signals Microsoft's bet that AI agents will expand beyond pure software into dedicated ambient hardware endpoints, potentially creating a new enterprise deployment category for distributed AI workloads running at the edge.
