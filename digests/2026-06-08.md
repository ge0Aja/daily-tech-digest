### CVE-2026-41089: Zero-Click Netlogon RCE Actively Exploited Against Domain Controllers
**Source:** BleepingComputer / SecurityWeek | [Read more](https://www.bleepingcomputer.com/news/microsoft/critical-windows-netlogon-remote-code-execution-flaw-now-exploited-in-attacks/)

A CVSS 9.8 stack-based buffer overflow in Windows Netlogon allows an unauthenticated, network-adjacent attacker to achieve SYSTEM-level code execution on domain controllers — no credentials, no user interaction required. Patched in May 2026 Patch Tuesday, public PoC code is available and active exploitation was confirmed May 29, prompting Belgium's Centre for Cybersecurity to issue a national warning. Every supported Windows Server version (2012–2025) is vulnerable; partial patching leaves remaining domain controllers exposed to lateral takeover. All domain controllers in an environment should be patched in a single maintenance window immediately.

**Source:** PostgreSQL Project | [Read more](https://www.postgresql.org/about/news/postgresql-19-beta-1-released-3313/)

**Source:** Apache Fluss | [Read more](https://fluss.apache.org/blog/fluss-open-source/)

Apache Fluss (now in Apache incubation) was open-sourced under Apache 2.0, providing columnar streaming storage with sub-second data freshness, primary-key table semantics, and native Apache Flink integration.
