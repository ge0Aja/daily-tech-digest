### Claude Code GitHub Action Flaw Enabled Supply-Chain Hijacking via Prompt Injection
**Source:** Microsoft Security Blog / GMO Flatt Security | [Read more](https://www.microsoft.com/en-us/security/blog/2026/06/05/securing-ci-cd-in-agentic-world-claude-code-github-action-case/)

Microsoft Threat Intelligence disclosed a prompt injection vulnerability in Anthropic's Claude Code GitHub Action that allowed attackers to exfiltrate CI/CD workflow secrets — including the `ANTHROPIC_API_KEY` — by embedding malicious instructions inside GitHub issues as HTML comments invisible to human reviewers but readable by the AI model. The Read tool had unsandboxed access to `/proc/self/environ`, bypassing environment variable scrubbing applied to the Bash tool. A separate supply-chain flaw (patched in v1.0.94, found by RyotaK at GMO Flatt Security) let unauthenticated external attackers push malicious code to any downstream repository using a flawed `checkWritePermissions` function. Anthropic patched the `/proc/` exposure in Claude Code v2.1.128 (May 5); teams using the GitHub Action should verify they are on the latest versions of both.

### CISA Orders Federal Agencies to Patch Actively Exploited SolarWinds Serv-U DoS (CVE-2026-28318)
**Source:** BleepingComputer / Help Net Security | [Read more](https://www.bleepingcomputer.com/news/security/cisa-hackers-now-exploit-solarwinds-serv-u-flaw-to-crash-servers/)

CISA added CVE-2026-28318 to its Known Exploited Vulnerabilities catalog on June 5, ordering US federal civilian agencies to patch by June 19. The unauthenticated DoS flaw (CVSS 7.5) in SolarWinds Serv-U is triggered by sending a malicious POST request with `Content-Encoding: deflate`, crashing the service with zero credentials required — no exploitation complexity beyond a crafted HTTP request. SolarWinds released a fix in Serv-U 15.5.4 Hotfix 1; organizations that cannot patch immediately should block POST requests containing `content-encoding` at the network perimeter, as Serv-U does not use that capability legitimately.
