### Miasma Worm Compromises 73 Microsoft GitHub Repos via AI Coding Tools
**Source:** The Hacker News / StepSecurity | [Read more](https://thehackernews.com/2026/06/miasma-worm-hits-73-microsoft-github.html)

On June 5, the Miasma worm — a self-replicating supply chain malware linked to threat group TeamPCP — hit 73 Microsoft GitHub repos across Azure, Azure-Samples, Microsoft, and MicrosoftDocs orgs. Attackers used previously stolen contributor credentials to push a malicious commit that embedded a credential-harvesting payload triggered when repositories are opened in AI coding tools including Claude Code, Gemini CLI, Cursor, and VS Code. The worm propagated using harvested cloud credentials; GitHub disabled all affected repos in an automated sweep lasting just 105 seconds, but the takedown broke CI/CD pipelines for organizations relying on Azure/functions-action.

### PostgreSQL 19 Beta 1 Released: Parallel Autovacuum, SQL/PGQ Graphs, Online Checksums
**Source:** PostgreSQL.org | [Read more](https://www.postgresql.org/about/news/postgresql-19-beta-1-released-3313/)

PostgreSQL 19 Beta 1 landed on June 4 with several operationally significant features: parallel autovacuum improves maintenance throughput on high-write tables; SQL/PGQ property graph queries add native graph query capabilities to relational workloads; and online checksum enable/disable eliminates the need for cluster restarts during DBA operations. Logical replication can now be promoted without a restart, and the new ON CONFLICT DO SELECT clause simplifies upsert patterns. JIT compilation is disabled by default and the default TOAST compression switches to LZ4; GA is expected around September/October 2026.
