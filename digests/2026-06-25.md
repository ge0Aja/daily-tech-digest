### Tencent Open-Sources TencentDB Agent Memory: 61% Token Reduction for Long-Horizon Agents

**Source:** MarkTechPost | [Read more](https://www.marktechpost.com/2026/05/23/tencent-open-sources-tencentdb-agent-memory-a-4-tier-local-memory-pipeline-for-ai-agents/)

Tencent released TencentDB Agent Memory under the MIT license, a four-tier local memory pipeline (L0 Conversation → L1 Atom → L2 Scenario → L3 Persona) designed to fix context bloat and recall failure in long-running AI agents. The backend is local SQLite with the sqlite-vec extension, requiring no external API calls or cloud dependencies. Benchmark data shows a 61.38% reduction in token consumption and a 51.52% improvement in task pass rate when integrated with the OpenClaw plugin. This is directly useful for teams shipping production agents that handle multi-turn, long-horizon workflows where context costs are significant.

## Security & Privacy

### CISA Flags Ubiquiti UniFi OS Triple Flaw Chain — Full Root RCE, Patch Deadline June 26

**Source:** BleepingComputer / CyberPress | [Read more](https://www.bleepingcomputer.com/news/security/cisa-warns-of-max-severity-ubiquiti-flaws-exploited-in-attacks/)

CISA added three Ubiquiti UniFi OS vulnerabilities (CVE-2026-34908, 34909, 34910) to its Known Exploited Vulnerabilities catalog after confirmed in-the-wild attacks. The trio chains improper access control → path traversal → command injection to achieve full root remote code execution; Bishop Fox confirmed the complete chain in a technical write-up and attackers have been observed creating rogue admin accounts named "John Sim" during automated reconnaissance. Patches are available in UniFi OS Server 5.0.8 (released May 21). Federal agencies must patch by June 26; all organizations running UniFi infrastructure should treat this as critical priority.

CISA added CVE-2026-48907 — a CVSS 10.0 improper access control vulnerability in the Widget Factory Joomla Content Editor (JCE) plugin — to the Known Exploited Vulnerabilities catalog, citing evidence of active exploitation enabling arbitrary PHP code execution. Amazon Aurora DSQL has reached general availability in 14 AWS regions with active-active multi-region architecture: no leader node, no failover events, no replication lag on commit — concurrent reads and writes across all regions with strong consistency. ## Systems & Engineering

### LeaseGuard: Raft Leases Done Right — Zero-Roundtrip Consistent Reads (SIGMOD 2026)

**Source:** ACM / arXiv | [Read more](https://dl.acm.org/doi/10.1145/3786663)

Researchers from MongoDB published LeaseGuard in SIGMOD 2026, a new Raft leader-lease algorithm that delivers strongly consistent reads with zero network roundtrips by anchoring the lease to log commit progress rather than clock drift. Prior lease protocols suffer from vague specifications and reduce availability during failover; LeaseGuard eliminates contention between writes and quorum reads, boosting write throughput from ~1,000 to ~10,000 ops/sec in testing while allowing a new leader to serve 99% of reads instantly rather than waiting for the full lease timeout. The algorithm is formally specified in TLA+ and has been open-sourced. Engineers building or evaluating distributed SQL/KV systems should review this before selecting a consistency mechanism.

**Source:** Global Policy Watch | [Read more](https://www.globalpolicywatch.com/2026/06/eu-ai-act-update-timeline-relief-targeted-simplification-and-new-prohibitions-2/)

EU negotiators reached a provisional agreement on the Digital Omnibus amendments to the AI Act on May 7, 2026, deferring obligations for Annex III high-risk AI systems from August 2, 2026 to December 2, 2027 — a 16-month extension. For engineering teams building AI systems deployed in Europe, this significantly extends the runway for the most operationally complex compliance requirements. General Purpose AI model obligations under Article 50 (machine-readable labeling) remain in effect from August 2026.
